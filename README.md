# エモリー（Emory）

日々の感情を **3Dの丸い感情ボール** として記録し、スマホの中に
**物理的に溜めていく** 感情ログアプリ。

> 「今日の気分をひとつ投げ込む」→「気づいたら自分の感情がスマホの中に溜まっている」
> →「スクロールすると過去の気持ちがぎっしり詰まって見える」

文章で内省する日記でも、感情を数値化する分析ツールでもなく、
感情ログを **触れる・溜まる・眺める** 体験にするアプリ。

## 体験

1. 今の気分に近い感情ボール（8種）を選ぶ
2. 画面上で **上方向にドラッグ／フリック** して投げ込む
3. ボールが画面下部に落ちて物理で積み重なる
4. 下にスクロールすると、過去の感情が日付ごとにギチギチに詰まって見える

## 技術スタック

- **Expo + React Native + TypeScript**（iOS / Android / Web）
- **matter-js** — 投擲と積み上げの物理演算
- **react-native-gesture-handler** — スリングショット式の投擲操作
- **react-native-svg** — ターゲット・軌道線・日付境界の描画
- **AsyncStorage** — 端末ローカル保存

## セットアップ

```bash
npm install
npm run web        # ブラウザで起動
npm run ios        # iOS シミュレータ
npm run android    # Android エミュレータ
npm run typecheck  # 型チェック
```
