# エモリー 挙動仕様書（デグレ防止用）

このファイルは「過去に合意した挙動」を明文化し、**デグレ（先祖返り）を防ぐ**ためのもの。
変更を加えるときは、ここの不変条件（INVARIANTS）を壊していないか必ず確認すること。
自動チェックは `scripts/check-spec.ts`（`npm run check` / CI で実行）。

## 体験の核（縦長の箱）
- 縦長の「箱」を正面から見ている。上から絵文字を投入し、下から積み上がる。
- 積むほど層が高くなり、カメラは上へ。下にドラッグで箱の底が見える。

## INVARIANTS（不変条件・絶対に守る）

1. **積層はランダム**（最重要・過去に2回デグレ）
   - 絵文字層は「整列したグリッド」ではなく「**ランダムに積み上がった**」見た目。
   - 実装: `computeSeedLayout` が各ボールに十分なジッター（横 `JITTER_X_RATIO=0.18`×径以上）を与える。
   - 自動検証: 配置の x 座標がグリッド値に揃わない（distinct 値・残差分散が閾値以上）。

2. **重ならない**
   - 着地状態で絵文字同士が重ならない。
   - 実装: 物理ボディ半径 = 見た目の半径（≈0.42×径）、`slop` 小、動的で物理が重なりを解消。

3. **下層は固定・上層は動的**
   - 山頂から `ACTIVE_DEPTH_ROWS`(=6) 径ぶんの上層は動的（衝突で反応・ジョストル）。
   - それより深い眠りボールは静的化（崩れ防止＋軽量）。着地後に勝手に「全固定」しない。

4. **カメラは着地済みのみ基準**
   - カメラの山頂判定は「着地済み（静的/眠り）」だけ。**落下中・跳ね中のボールは追わない**。
   - リロード時も含め、絵文字層は常に画面内に収まる（帯 `[0.34, 0.72]` を外れたら中央へ）。

5. **投擲＝スリンガーのみ**
   - ドラッグ＝スリンガー（引いて離す）。フリック/長押しモードは無い。
   - 射出後 0.5s は次の待機ボールを出さない（飛翔を見せる）。

6. **UI は飛翔を隠さない**
   - 感情ピッカーは半透明オーバーレイ。一言メモUIは無い。

7. **UFO**
   - 画面外を左右に往復。命中で花火＋消滅→再出現。当たっても絵文字は箱に落ちて積もる。

## 調整パラメータ（壊さず数値だけ動かす想定）
- 弾性: `useBoxPhysics` の `restitution`
- 上層の厚み: `ACTIVE_DEPTH_ROWS`
- カメラ帯: `TOP_LIMIT_FRAC / BOTTOM_LIMIT_FRAC / TARGET_FRAC`（AddScreen）
- ランダム量: `JITTER_X_RATIO / JITTER_Y_RATIO`（seedLayout）

## デグレ・チェックリスト（手動確認）
- [ ] リロード直後、絵文字層が画面内に見える（落ち切って消えない）
- [ ] 絵文字層がランダム（グリッド整列していない）
- [ ] 絵文字同士が重なっていない
- [ ] 投げると上層が反応して動く／下層は崩れない
- [ ] 飛ばした絵文字がUIの裏を通って上まで見える
